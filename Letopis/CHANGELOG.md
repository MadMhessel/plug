# Changelog

## 1.0.1
- Шкалы переведены на дробные значения для точных начислений.
- Экспорт JSON переносится в папку `export/` с датой и временем в имени файла.
- БД хранит значения шкал как REAL, добавлено сохранение времени опасных событий.

## 1.0.0
- Реализованы шкалы напряжения с лимитами и спадом.
- Добавлены предвестники, BossBar и события с мини-боссами.
- Реализован алтарь Летописи с ритуалами.
- Добавлены трофеи, печати, титулы и косметика.
- Команды игрока и администратора, журнал и экспорт JSON.
