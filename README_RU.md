# Letopis — стартовый репозиторий

Это **заготовка** Paper-плагина под Minecraft 1.20.1+ (JDK 17, без NMS).
Внутри: базовая структура проекта, ресурсы `config.yml` и `messages.yml`, схема SQLite.

## Сборка
### Вариант 1 — IntelliJ IDEA
1. Откройте папку проекта как Gradle-проект.
2. Дождитесь загрузки зависимостей.
3. Выполните задачу Gradle: `build`.
4. Готовый файл: `build/libs/Letopis-0.1.0.jar`.

### Вариант 2 — через Gradle
Если Gradle установлен:
```bash
gradle build
```

## Установка на сервер
1. Положите `Letopis-0.1.0.jar` в папку `plugins/`.
2. Запустите сервер.
3. Проверьте, что появились:
   - `plugins/Letopis/config.yml`
   - `plugins/Letopis/messages.yml`
   - `plugins/Letopis/letopis.db`

## Команда
- `/letopis` — пока выводит сообщение о том, что это заготовка.
